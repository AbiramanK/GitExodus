# Contributing to GitExodus

Welcome to GitExodus! We're excited that you want to contribute. This project aims to make Git repository management simple and efficient.

## Ways to Contribute

1.  **Reporting Bugs**: If you find a bug, please open an issue using the Bug Report template.
2.  **Suggesting Enhancements**: We're always looking for ways to improve. Use the Feature Request template for new ideas.
3.  **Submitting Pull Requests**:
    *   Fork the repository.
    *   Create a feature branch: `git checkout -b feat/my-new-feature`.
    *   Commit your changes: `git commit -m 'feat: add some amazing feature'`.
    *   Push to the branch: `git push origin feat/my-new-feature`.
    *   Open a Pull Request.

## Code Standards

*   **Rust**: Use `cargo fmt` and `cargo clippy`.
*   **TypeScript**: Follow the existing project patterns. We use ESLint and Prettier.
*   **Commits**: Use [Conventional Commits](https://www.conventionalcommits.org/).
