# Security Policy

## Reporting a Vulnerability

We take the security of GitExodus seriously. If you believe you have found a security vulnerability, please report it to us responsibly.

**Please do not report security vulnerabilities via public GitHub issues.**

Instead, please send an email to [INSERT EMAIL ADDRESS]. 

We will acknowledge receipt of your report within 48 hours and provide a timeline for addressing the issue.

## Supported Versions

Only the latest stable release of GitExodus is supported for security updates.
